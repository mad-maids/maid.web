## Purposeful Animations

![](/assets/blog/0_Src2uyUU6zQFaGsB__1_.gif)

Thanks to a multitude of JavaScript frameworks and libraries, along with new browser capabilities. Creating well thought out animations within the website DOM has never been easier. This undoubtfully has made this the biggest trend within web design and development. This is nothing new to the game but it is being used more and more with extensive attention to detail. Quite frankly motion within a website gives life to the content being displayed.

## 3D Rendering

![](/assets/blog/image_processing20190918-2159-1tzca9b-1400x1050.jpg)

There are some websites out there that are adopting 3D design rendering for that hero section either as a fun design motif or for showing what they currently offer within there company. Various examples lend themselves to moving objects or static renditions.

## Color & Gradients

![](/assets/blog/Spotify-2015-2-1400x713.jpg)

With screens on devices getting more and more powerful allowing better and cleaner rendition of color. We as designers or developers have seen many take advantage of the use of color. More and more sites are adopting heavy use of vibrant color as well as the use of clean gradients. For instance, big names like Apple and Spotify sometimes use gradients within there site.
